## 결론

- **G4 게이트(≥10× 향상)**: BVH N=100에서 이미 **13.7×** 달성. N이 커질수록 우위 증가.
- Release 빌드(`maturin build --release`) 시 추가 향상 예상.
- GJK/PGS는 dev build 기준이며 release에서 SIMD 최적화가 적용됨.
